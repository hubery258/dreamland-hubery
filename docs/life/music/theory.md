### 调与和弦

`1=C`等价于说是C大调（大调全称自然大**调式**），大调是:**全全半全全全半**,调中七个音名按顺序出现。小调是以la作为起手，全半全全半全全。

其中存在**关系大小调**(*平行大小调*):C大调与A小调是关系大小调关系，找到关系大调即可找到小调，要找到小调音阶先找高一个全音一个半音后的大调，然后列出关系大调，67挪位成12即变为小调

音程: 两个音隔几个钢琴上的白键(包含二者自身)就是几度，先还原数白键得度，再直接按键盘来数格子对应得属性
![音程属性](https://s41.ax1x.com/2026/02/03/pZ53cVI.png)

以C大调为例学习音名,以谁为根音即为什么和弦，而以属音为根音的又可以叫某调的属和弦。
![乐名](https://s41.ax1x.com/2026/02/03/pZ532IP.jpg)

#### 三和弦

以三个音(根音，三音，五音)按**三度关系**叠加的和弦:
1. 大三和弦: 大三度 + 小三度 (C和弦:CEG)
2. 小三和弦: 小三度 + 大三度 (Cm: C bE G)
3. 减三和弦：小三度 + 小三度 (Cdim: C bE bG)
4. 增三和弦：大三度 + 大三度 (Caug: C E #G)

在C大调中，其中非离调三和弦(和弦中没有离调音，即不在原调的音阶中的音):
![C大调](https://s41.ax1x.com/2026/02/03/pZ58iIx.md.jpg)

#### 七和弦

四个音按三度关系叠加的和弦

1. 大七和弦：大三和弦 + 大三度 (Cmaj7: C E G B)
2. 大小七和弦：大三和弦 + 小三度 (C7: C E G bB)
    - 在F调中，C是属音，那么C7就变成了属七和弦
    - 在C调中G7(G B D F)就是属七和弦
    - 每个调只有一个属音，就只有一个属七和弦，其他的都是大小七和弦，且除了属七和弦，其他大小七和弦都有离调音
3. 小七和弦: 小三和弦 + 小三度 (Cmin7: C bE G bB)
4. 小大七和弦：小三和弦 + 大三度 (CmM7: C bE G B)
5. 减七和弦： 减三和弦 + 小三度 (Cdim7: C bE bG bbB重降)
6. 半减七和弦：减三和弦 + 大三度 (Cm7-5:C bE bG bB)

#### 九和弦

五个音按三度关系叠加的和弦

1. 大九和弦: 大七和弦 + 小三度 (Cmaj9: C E G B D)
2. 小九和弦: 小七和弦 + 大三度 (Cm9: C bE G bE D)
3. 属九和弦: 属七和弦 + 大三度 (C9: C E G bB D就是F调的属九和弦)
4. 属七降九和弦：属九和弦的**九音降半音** (C7b9: C E G bB bD)
5. ADD9和弦：三和弦 + 九音 (Cadd9: C E G D)

大x和弦与小x和弦就是大+小+大...一直组合，大x和弦从大开头，小x和弦从小开头

#### 挂留，转位和弦

1. 挂二和弦: 用二度音换掉原本的三音 (Csus2: C D G)
2. 挂四和弦: 四度音换三音 (Csus4: C F G)
!!! tip inline end
    挂留和弦一般用于调式的1级和弦、5级和弦

3. 转位和弦: 根音不在最低音的和弦 
    - G/B: B D G, B是三音同时为最低音，第一转位
    - G/D: D G B, D是五音同时为最低音，第二转位
    - 七和弦就有第三转位，无论是什么转位，根音不变，提高八度。
