# PM相关经验总结

在项目管理（PM）的宏观层面，**第一步要做的事情就是“明确方向”**。具体来说，它包含三个核心动作：
### 🧭 PM 第一阶段的核心任务
1. **定义目标（Goals）**  
   - 项目要解决什么问题？  
   - 最终要达到什么状态？  

2. **明确价值（Value）**  
   - 为什么值得做？  
   - 做成后对个人、团队或未来有什么意义？  

3. **形成愿景（Vision）**  
   - 把目标和价值浓缩成一句话或一段话，作为项目的“北极星”，让后续所有决策有参照。  

换句话说，PM 的第一步就是：**先回答“为什么做”和“要做到什么”，而不是“怎么做”**。
